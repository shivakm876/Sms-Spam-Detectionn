# Sms-Spam-Detectionn

This project identifies and filters spam messages using machine learning by analyzing SMS content. It involves data collection, preprocessing (lemmatization, TF-IDF vectorization, Bag of Words, stop word removal), feature engineering, and model selection. The dataset is from KimDongH/Spam_Dataset.
